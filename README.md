# QuickTweet
  A Social Media App 
</div>

## **How to install and run locally ?**

```
$ git clone https://github.com/yash-purkar/QuickTweet.git
$ cd QuickTweet
$ npm install
$ npm start
```
## **Features -**

- Feed Posts
- Explore & Trending Posts
- Add, Update & Delete Post
- Add & Delete Comment
- User Profile Update
- Follow/Unfollow
- Search User
- Like and Comment on Post
- Share post
- Authentication:
  - User Signup
  - User Login
  - User Logout
 - Fully Responsive

## **Built with -**

- React JS
- useReducer  & useContext Toolkit 
- React Router v6
- vanilla css
